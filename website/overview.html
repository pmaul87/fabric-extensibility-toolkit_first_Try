Overview

Lineage Workbench

Lineage Workbench is a custom Fabric workload item that helps teams extract technical lineage, inspect dependencies, and track implementation requirements in one place.

Capabilities

What the app does

Guided setup

Deploy the Lakehouse, Environment, and Notebook assets needed for extraction.

Automated extraction

Collect lineage data from supported Fabric artifacts across selected workspaces.

Interactive exploration

Inspect dependency graphs, detail panels, and upstream or downstream relationships.

Requirement tracking

Attach requirement tickets directly to lineage nodes for planning and governance work.

Included

Repository contents

  • Setup wizards for onboarding the workload
  • Extraction notebooks for lineage collection
  • Lineage graph views and detail-side analysis
  • Integrated requirement ticket management
