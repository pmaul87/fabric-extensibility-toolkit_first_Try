# Release Notes - v2026.07

## Overview

This branch consolidates the Lineage Workbench extraction flow, notebook deployment, and publish packaging for the current Fabric workload workstream.

## Changes

### Lineage Workbench updates
- Added a deployment wizard for the extraction notebooks and orchestration pipeline.
- Updated the workspace selection flow to capture workspace types and warn on likely shared or Pro-only workspaces.
- Simplified the extraction flow to pass only the selected target workspaces into the pipeline trigger.

### Notebook pipeline updates
- Converted the extraction notebooks to lazy loading where appropriate.
- Fixed Notebook 1 parameter handling so injected values are not overwritten by notebook defaults.
- Updated the notebook orchestration pipeline payload shape to use the `TridentNotebook` `parameters` contract.

### Packaging and publishing
- Updated manifest packaging so the current item manifest is included in the publish package.
- Aligned production/test environment item names with the current `LineageWorkbench` item.
- Confirmed the manifest package builds successfully for the production environment.

## Validation

- Package output includes both the workload manifest and the current item manifest files.
