Getting Started

Setup and local run

These are the essentials from the project README for getting the workload running in a Fabric development environment.

Prerequisites

Before you begin

  • Node.js
  • PowerShell 7
  • .NET SDK
  • VS Code
  • Microsoft Fabric tenant and workspace
  • Microsoft Entra app

Commands

Setup steps

1

Run setup

Initialize the workload configuration and supporting resources.

cd scripts\Setup
.\Setup.ps1 -WorkloadName "Org.InsightWorkbench"
2

Start development mode

Run the local dev server and gateway, then open the workload in Fabric Workload Hub.

cd scripts\Run
.\StartDevServer.ps1
.\StartDevGateway.ps1
